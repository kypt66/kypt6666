# SciFigure - 免费科研组图编辑器

SciFigure 是一个面向科研论文 Figure 排版的在线组图工具，适合快速制作 Nature、Science、Cell 风格的多子图排版。

## 产品定位

- 功能全开，不做注册墙或 Pro 付费墙
- 通过流量广告、科研服务赞助和联盟广告持续运营
- 广告位不进入导出画布，不影响科研图片成品

## 核心功能

- 期刊级组图模板
- 图片上传、拖拽移动、缩放和填充/适应切换
- a/b/c、A/B/C、数字、罗马数字子图标签
- 总标题编辑和拖拽定位
- 智能吸附辅助线
- 2x、3x、4x PNG 高清导出
- 反馈与广告合作入口
